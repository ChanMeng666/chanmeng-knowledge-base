## What's Inside

This knowledge base documents practical solutions and lessons learned across full-stack development, covering:

- **Authentication** — Neon Auth + Flask integration, email verification & password reset flows
- **Payments** — Stripe + Cloudflare Workers end-to-end payment integration
- **AI Integrations** — CopilotKit + LangGraph HITL workflows, Notion API as FAQ backend
- **Architecture** — Event-driven plugin architecture (Claude Code Audio Hooks)
- **Frontend** — Canvas game mobile optimization, SVG animation generation techniques
- **SEO** — Generative Engine Optimization (GEO) for AI search engines
- **DevOps** — Railway → DigitalOcean migration, YouTube + GitHub Actions automation

## Tech Stack

Built with [Quartz 4](https://quartz.jzhao.xyz/) — a fast, batteries-included static site generator that transforms Markdown into a fully functional website. Content is authored in [Obsidian](https://obsidian.md/) and deployed via GitHub Pages.

## Local Development

```bash
git clone https://github.com/ChanMeng666/chanmeng-knowledge-base.git
cd chanmeng-knowledge-base
npm i
npx quartz build --serve
```
